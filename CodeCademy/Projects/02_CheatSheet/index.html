HTML TAGS

<html>...</html>

The tag shows up at the beginning and end of an HTML document (known as the root element). It indicates that the webpage is written in HTML5, and all other page markup comes in between these beginning and ending tags.

<head>...</head>

The head tag contains information specific to the page, including the title tags, meta data, and links to scipts and stylesheets.

<title>...</title>

Contains the title for that page. It is useful for both search engines (when scanning and indexing pages) and users by showing up on the browser's tab.

<body>...</body>

Contains all the content that will be shown to the users, including everything they'll see and read.
